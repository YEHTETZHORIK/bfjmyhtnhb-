Java Programming Language

Java သည် Object-Oriented Programming Language တစ်ခုဖြစ်ပြီး Application Development နဲ့ Software Development တွေမှာ အရမ်းအရေးကြီးပါတယ်။ Java ကို Platform-independent Programming Language အနေနဲ့ လူသိများပြီး "Write Once, Run Anywhere" ဆိုတဲ့ စကားလုံးနဲ့ ရည်မှန်းဖန်တီးထားပါတယ်။

Java ရဲ့ သမိုင်းကြောင်း

၁၉၉၅ ခုနှစ်မှာ Sun Microsystems ရဲ့ James Gosling က Java ကို ဖန်တီးခဲ့ပြီး အစပိုင်းမှာ "Oak" ဆိုတဲ့နာမည်ကို အသုံးပြုခဲ့တယ်။ နောက်ပိုင်းမှာ Java လို့ပြောင်းလဲပြီး Programming World အနောက်မှာ အကျော်ကြားဆုံး Language တစ်ခု ဖြစ်လာခဲ့တယ်။

Java ရဲ့ အင်္ဂါရပ်များ

  • Platform-independent - "Write Once, Run Anywhere"
  • Object-Oriented Programming Language
  • Security Features များစွာ ပါဝင်သည်
  • Large Community Support
  • အခြေခံ Application တွေမှ Enterprise-level Application အထိ ဖန်တီးနိုင်

Java ရဲ့ နမူနာများ

// Java အခြေခံ နမူနာ
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("မင်္ဂလာပါ Java");
    }
}

// Function တစ်ခုရေးမယ်
public class SumExample {
    public static int addNumbers(int a, int b) {
        return a + b;
    }
    public static void main(String[] args) {
        int result = addNumbers(10, 20);
        System.out.println("အဖြေက " + result);
    }
}

// Object-Oriented နမူနာ
class Animal {
    String name;

    // Constructor
    public Animal(String name) {
        this.name = name;
    }

    // Method
    public void speak() {
        System.out.println(name + " ပြောဆိုနေသည်။");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal dog = new Animal("ခွေး");
        dog.speak();
    }
}
